# Screenshots

[../../README.md](../../README.md) で参照されている、Misskey 上の note プレビューカードのスクリーンショット置き場。

| ファイル名 | 想定する内容 |
|---|---|
| `misskey-card-gigazine.png` | GIGAZINE 記事など、サムネイル付きの一般的なリンクプレビューカード |
| `misskey-card-wikipedia.png` | Wikipedia 記事のリンクプレビューカード（`wikipedia` プラグイン経由で生成） |
| `misskey-card-twitter-collapsed.png` | X (Twitter) ポストの折りたたみ表示プレビューカード |
| `misskey-card-twitter-detail.png` | X (Twitter) ポストの展開表示（`Read more on X` ボタン付きのフル埋め込み） |

## 追加・差し替え時の注意

- Misskey で note にリンクを貼ったときに表示されるカード UI を切り抜いて配置
- ファイル名・拡張子は README で参照しているものに合わせる
- 機微な情報（DM URL・個人情報・未公開アカウント等）が映り込まないものを使う
- 画質を保ちつつもサイズを抑える（リポジトリの `git clone` サイズに乗るため、目安: 各 100 KB 以下）
