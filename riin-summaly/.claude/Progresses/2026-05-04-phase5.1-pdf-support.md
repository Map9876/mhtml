## 実装方針の変更

計画では `pdf-parse(buffer, { max: 1 })` で 1 ページのみパースする方針だったが、pdf-parse v2 の API は v1 と全く異なる。代わりに `new PDFParse({ data }).getInfo()` を使用。`getInfo()` は document-level metadata のみ読むため「1 ページのみパース」より安全な実装になった (ページ解析は走らない)。
